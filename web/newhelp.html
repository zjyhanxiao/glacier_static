常见问题

Q : 为什么客户在美信的平台上投资产品是安全的？如果网站发生故障，客户投资产品的资金和回报会不会受影响？

A : 美信网站采用高标准安保措施，严格保护客户个人信息。另外，网站只是客户浏览美信平台上的产品以及完成初始投资流程的线上界面，网站发生的故障不会影响产品本身的存续。目前美信发行的安心计划系列里的产品全部以专属私募基金（Special Purpose Vehicle）架构模式搭建。客户持有专属基金的股权，通过该股权享受底层资产收益权。只要产品正常存续，即使没有网站，客户也将如期获得预计本息。

Q : 什么人可以投资安心计划？

A : 美国合格投资人，非美国（国际）投资人。

Q : 在平台上完成投资需要哪些步骤？

A : 您首先需要注册一个美信账号。登录后查看平台上开发的项目。选中感兴趣的项目，点击投资后填写投资基本信息、选择付款方式、在线签署法律文件。投资完成后，您可以在“我的账户”里查看投资详细信息和项目动态。因为本平台的项目均为美元投资项目，如果您目前没有美元存款，您可能在付款前需要在本地银行换汇。

Q : 若基金管理人（美信）破产对基金投资人有何影响？

A : 美信作为基金管理人破产将不对基金投资人产生实质的经济利益影响。若基金管理人破产，该基金将指定其他合格管理人，维护基金存续。

Q : 大陆居民如何投资在美国的美元产品？

A : 中国大陆居民需要用电汇（wire）的方式划款。款项可以来自中国大陆的美元银行账户或位于香港的离岸美元银行账户。由中国大陆的美元银行账户汇出的款项，其额度及用途请参照国家外汇管理局相关规定。在美国拥有银行账户的中国大陆居民可以同美国居民一样选择ACH网上转账、支票以及电汇三种方式划款。

Q : 如何在大陆开美元离岸账户？

A : 部分银行提供在大陆网点远程开通香港离岸账户服务，例如招商银行的香港一卡通。

Q : 客户投资的款项进入到什么账户？

A : 客户的款项会直接进入对应项目的托管银行账户。筹款结束后，资金不经由平台，将直接进入项目。每个项目有各自的托管银行账户，相互独立，保证专款专用。

Q : 项目的收益和本金如何返回？

A : 项目的收益和本金默认将返回到投资人注资使用的银行账户。如果客户需要以不同的账户收款，需要和我们的客服人员及时联系。此外，因中国外汇管理政策原因，存在资金无法直接返还到大陆银行的个人账户的风险。届时投资人需要提供一个离岸账户接受款项。
