美信金融是谁？

美信金融是由贝塔斯曼亚洲基金和梅花创投投资的海外理财互联网金融公司，由累计50年以上海外投行和资管经验的团队于2015年在纽约联合创立，致力于用先进互联网技术和金融创新使海外理财和全球资产配置更透明、更高效，成为大众触手可及的理财方式之一。创新理财产品“安心计划”专注于安全资产，让投资人轻松理财并获得可预期的稳定回报。
